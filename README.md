# ion flutter app

## Getting Started

Make sure your flutter is `2.0`.

- `./scripts/project_tools.sh create`

### For Android/iOS

- `flutter run`

### For macOS

- `flutter run -d macos`

### For Web

- `flutter run -d chrome`
